# דתיים שלובים · הירוקה בעופרים 🌿

מיניסייט הומוריסטי המבוסס על [חוק דתיים שלובים](https://he.wikipedia.org/wiki/%D7%97%D7%95%D7%A7_%D7%93%D7%AA%D7%99%D7%99%D7%9D_%D7%A9%D7%9C%D7%95%D7%91%D7%99%D7%9D) — התיאוריה שבין כל שני דתל״שים יש לכל היותר מכר אחד משותף.

עונים על כמה שאלות (ישיבה, שבט בבני עקיבא, מכינה/ש״ש, איפה אשתך עשתה שירות, ואיזה מעיין בצפון) — והאתר מחשב לכם את **מדד השלובוּת**: לכמה תושבים ב"הירוקה בעופרים" אתם כבר מחוברים.

הכל בשפה של "הירוקה בעופרים", עם קונפטי בסוף. אתר סטטי לגמרי (HTML/CSS/JS), הנתונים לא נשמרים בשום מקום.
